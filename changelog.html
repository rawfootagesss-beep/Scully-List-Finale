System Log

Registro de actualizaciones de la arquitectura OMNI

V25.3 - MODERATION ENGINE

Nuevo Panel de Moderación y Webhooks

  • Rango Moderador (Mod): Se agregó la jerarquía "Mod" para delegar tareas de seguridad sin arriesgar la Base de Datos. Integración de panel exclusivo (mod.html).
  • Integración "wjos": Nuevo moderador oficial añadido a la cabecera del Staff en el index.
  • Triple Webhook Discord: El panel Admin ahora soporta 3 canales de alertas distintos (General, Sugerencias y Niveles).
  • Notificaciones en Vivo: El sistema enviará Webhooks automáticos al añadir, editar o borrar niveles, cambiar el ranking o procesar sugerencias.
  • Traductor Nativo: Se implementó la API de Google Translate con un botón sutil (EN/ES) en el Hub Principal que traduce datos estáticos y dinámicos (chat/mercado) en tiempo real.
V25.2 - COSMETIC HUB

Motor Universal de Cosméticos

  • Tienda sin imágenes: Se eliminó la dependencia de enlaces Imgur en la tienda. Ahora se generan íconos dinámicos en base al nombre del efecto.
  • Precios Ajustados: Nuevos cosméticos económicos añadidos (Efecto Fantasma, Hielo, Fuego).
  • Sincronización Total: Los cosméticos equipados ahora se reflejan automáticamente en el Chat Global, el Buscador del Index y el Ranking de Titanes.
  • Bóveda Mejorada: Botones gigantes en el Perfil (Verde/Rojo) para hacer la equipación de ítems más amigable en celulares.
  • Animación Rainbow: Nueva lógica CSS integrada en todas las páginas para procesar el cosmético de Nick Arcoíris.
V25.1 - ABSOLUTE ENGINE

Smart Submit & Códigos

  • Smart Submit: La caja de sugerencias ahora extrae automáticamente el ID del video de YouTube y previsualiza la miniatura antes de enviar.
  • Fijación de Bugs de Letras: El canje de códigos secretos ahora reconoce mayúsculas y minúsculas (Ej: bLoOdlUst funcionará igual que BLOODLUST).
  • Root Access: Se simplificó el protocolo de emergencia de pantalla negra, usando un código rápido (score123).
  • Limpieza Visual: Se eliminaron las fotos de perfil del chat global para mantener una estética más limpia, tecnológica y futurista.
